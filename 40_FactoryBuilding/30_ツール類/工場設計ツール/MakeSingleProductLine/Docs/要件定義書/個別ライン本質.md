# 個別ライン本質 要件定義書

## 概要

個別ラインの自動設計を行うための必要な情報を保持しておくためのデータ


## 値

- 個別製造ライン名
- 使用レシピ名
- 使用レシピ数


## 値の説明

### 個別製造ライン名

個別製造ラインの名前。同工場の範囲内で一意の物。

個別製造ライン設計書の記載のほか、全体製造ライン設計にも使用される。


### 使用レシピ名

個別製造ラインで使用されるレシピの名前

個別ライン設計書の記載のため、レシピデータの取得の際にも使用される。


### 使用レシピ数

個別製造ラインで使用されるレシピの数。

個別ライン設計書の記載のため、レシピデータの input output 数と掛け合わせて使用される。
