# 個別ラインデータ 要件定義書

## 概要
個別ラインを自動設計し算出されたデータ。  
主に個別ライン設計書の自動記述に使用される。


## 値

- 個別製造ライン名
- 使用レシピ名
- 使用レシピ数
- 設備名
- 合計消費電力
- 
- 要求物品名1
- 要求物品単数1
- 要求物品総数1
- …
- 要求物品名4
- 要求物品単数4
- 要求物品総数4

- 作成物品名1
- 作成物品単数1
- 作成物品総数1
- …
- 作成物品名4
- 作成物品単数4
- 作成物品総数4


## 値の説明

### 個別製造ライン名

個別製造ラインの名前。同工場の範囲内で一意の物。

個別製造ライン設計書の記載のほか、全体製造ライン設計にも使用される。


### 使用レシピ名

個別製造ラインで使用されるレシピの名前

個別ライン設計書の記載のため、レシピデータの取得の際にも使用される。


### 使用レシピ数

個別製造ラインで使用されるレシピの数。

個別ライン設計書の記載のため、レシピデータの input output 数と掛け合わせて使用される。


### 設備名

使用する設備の名前


### 合計消費電力

設備の消費電力を合計した値。


### 要求物品名n

作成に際して使用する物品の名前

1 <= n <= 4 

要求物品がない場合、項目の記載を省略する。例として、要求する物品が2つであれば、要求物品名3,4 の項目は key, value 共に記載しない


### 要求物品単数1

作成に際して使用する物品の、レシピ単体の要求量


### 要求物品総数1

作成に際して使用する物品の、総数。単数にレシピ数をかけたもの。
