# 個別製造ライン設計書 自動ワイヤー製造ライン

## 概要
自動ワイヤーを製造するライン

## Input
- 固定子 10.0/m
- ケーブル 200/m

## Output
- 自動ワイヤー 10.0/m

## 必要設備
- 工作機 4ケ
- ベルトコンベア類 適量


## 製造ライン
```mermaid
flowchart TD

subgraph Input
    固定子([固定子])
    ケーブル([ケーブル])
end

工作機1[
    工作機1
    固定子 2.5/m
    ケーブル 50/m
    ↓
    自動ワイヤー 2.5/m
]

工作機2[
    工作機2
    固定子 2.5/m
    ケーブル 50/m
    ↓
    自動ワイヤー 2.5/m
]

工作機3[
    工作機3
    固定子 2.5/m
    ケーブル 50/m
    ↓
    自動ワイヤー 2.5/m
]

工作機4[
    工作機4
    固定子 2.5/m
    ケーブル 50/m
    ↓
    自動ワイヤー 2.5/m
]

subgraph Output
    自動ワイヤー([自動ワイヤー])
end

固定子-->|2.5|工作機1
ケーブル-->|50|工作機1
工作機1-->|2.5|自動ワイヤー
固定子-->|2.5|工作機2
ケーブル-->|50|工作機2
工作機2-->|2.5|自動ワイヤー
固定子-->|2.5|工作機3
ケーブル-->|50|工作機3
工作機3-->|2.5|自動ワイヤー
固定子-->|2.5|工作機4
ケーブル-->|50|工作機4
工作機4-->|2.5|自動ワイヤー
```
