# 個別製造ライン設計書 (製造ライン名)製造ライン

## 概要
初期一時工場の鉄鉱石から鉄のインゴットを製錬するライン

## Input
- 鉄鉱石 30/m

## Output
- 鉄インゴット 30/m
    - 鉄板ライン行き 30/m

## 必要設備
- 製錬炉 1
- ベルトコンベア類 適量


## 製造ライン
```mermaid
flowchart TD

subgraph Input
    鉄鉱石([鉄鉱石])
end

製錬炉1[
    製錬炉1
    鉄鉱石 30/m
    ↓
    鉄のインゴット 20/m
]

subgraph Output
    鉄板([鉄板])
end


鉄鉱石-->|30|製錬炉1
製錬炉1-->|20|鉄板


```
