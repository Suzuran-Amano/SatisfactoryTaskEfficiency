# 個別製造ライン設計書 鉄のインゴット製造ライン

## 概要
鉄鉱石から鉄のインゴットを製錬するライン

## Input
- 鉄鉱石 60/m

## Output
- 鉄のインゴット 60/m

## 必要設備
- 製錬炉 2ケ
- ベルトコンベア類 適量


## 製造ライン
```mermaid
flowchart TD

subgraph Input
    鉄鉱石([鉄鉱石])
end

製錬炉1[
    製錬炉1
    鉄鉱石 30/m
    ↓
    鉄のインゴット 30/m
]

製錬炉2[
    製錬炉2
    鉄鉱石 30/m
    ↓
    鉄のインゴット 30/m
]

subgraph Output
    鉄のインゴット([鉄のインゴット])
end

鉄鉱石-->|30|製錬炉1
製錬炉1-->|30|鉄のインゴット
鉄鉱石-->|30|製錬炉2
製錬炉2-->|30|鉄のインゴット
```
