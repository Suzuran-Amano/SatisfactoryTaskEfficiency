# 個別製造ライン設計書 ローター製造ライン

## 概要
以下の物品を生産する

- ローター

## Input
- 鉄のロッド 10/m
- ネジ 50/m

## Output
- ローター 8/m

## 必要設備
- 組立機 2ケ
- ベルトコンベア類 適量


## 製造ライン
```mermaid
flowchart TD

subgraph Input
    鉄のロッド([鉄のロッド])
    ネジ([ネジ])
end

組立機1[
    組立機1
    鉄のロッド 5/m
    ネジ 25/m
    ↓
    ローター 4/m
]

組立機2[
    組立機2
    鉄のロッド 5/m
    ネジ 25/m
    ↓
    ローター 4/m
]

subgraph Output
    ローター([ローター])
end

鉄のロッド-->|5|組立機1
ネジ-->|25|組立機1
組立機1-->|4|ローター
鉄のロッド-->|5|組立機2
ネジ-->|25|組立機2
組立機2-->|4|ローター
```
