# 個別製造ライン設計書 シリカ製造ライン

## 概要
シリカを製造するライン

## Input
- 未加工石英 22.5/m

## Output
- シリカ 37.5/m

## 必要設備
- 製作機 1ケ
- ベルトコンベア類 適量


## 製造ライン
```mermaid
flowchart TD

subgraph Input
    未加工石英([未加工石英])
end

製作機1[
    製作機1
    未加工石英 22.5/m
    ↓
    シリカ 37.5/m
]

subgraph Output
    シリカ([シリカ])
end

未加工石英-->|22.5|製作機1
製作機1-->|37.5|シリカ
```
