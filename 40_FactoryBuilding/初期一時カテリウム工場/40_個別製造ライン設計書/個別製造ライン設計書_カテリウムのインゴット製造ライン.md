# 個別製造ライン設計書 カテリウムのインゴット製造ライン

## 概要
以下の物品を生産する

- カテリウムのインゴット

## Input
- カテリウム鉱石 45/m

## Output
- カテリウムのインゴット 15/m

## 必要設備
- 製錬炉 1ケ
- ベルトコンベア類 適量


## 製造ライン
```mermaid
flowchart TD

subgraph Input
    カテリウム鉱石([カテリウム鉱石])
end

製錬炉1[
    製錬炉1
    カテリウム鉱石 45/m
    ↓
    カテリウムのインゴット 15/m
]

subgraph Output
    カテリウムのインゴット([カテリウムのインゴット])
end

カテリウム鉱石-->|45|製錬炉1
製錬炉1-->|15|カテリウムのインゴット
```
